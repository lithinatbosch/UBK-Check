# UBK Check

## Overview
The **UBK Check** repository contains scripts and resources for automating the UBK check process. This project is designed to streamline and simplify the UBK validation workflow.

## Folder Structure
```
01 User Guide/
02 Scripts/
    Release_UBK_Check.ps1
    UBKCheck.ps1
```

### Description of Files
- **01 User Guide/**: Contains documentation and user guides for the UBK check process.
- **02 Scripts/**: Contains PowerShell scripts for performing the UBK check.
  - `Release_UBK_Check.ps1`: Script for releasing the UBK check.
  - `UBKCheck.ps1`: Core script for executing the UBK check.

## Usage
1. Clone the repository to your local machine.
2. Navigate to the `02 Scripts` folder.
3. Execute the required PowerShell script using the following command:
   ```powershell
   .\<script_name>.ps1
   ```
   Replace `<script_name>` with the name of the script you want to run.

## Prerequisites
- Windows operating system.
- PowerShell v5.1 or higher.
